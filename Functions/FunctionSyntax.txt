type может быть любой существующий тип данны: 
bool, char, int, ...., или void - пустота.

---------------------------------------------------
Параметры - это обычные переменные, объявленные в заголовке функции.
Эти переменные являеются локальными для функции, то есть, существуют 
только в этой функции, только в ее области видимости, и не доступны 
за ее пределами.

Функция Power() возводит число в степень и должна знать 
какое число, и в какую степнь нужн возвести, соответсвенно, 
она будет принимать основание и показатель степени, и возвращать степень числа;
int Power(int a, int n);//a - основание степени, n - показатель степени.

Функция Print() выводит массив на экран. Она должна знать какой массив нужно вывести,
и сколько в нем элементов.
void Print(int arr[], const int n);//arr - массив, n - размер массива

Функция CoffeMachine() возвращает значение типа CupOfCoffe, принимает молотый кофе и воду.
